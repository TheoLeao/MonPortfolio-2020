Un projet web

Application web permettant de créer et de visualiser sur une map le parcours d’un barathon avec ReactJS avec JSX + Api en NodeJS

Dans le cadre de ma formation licence professionnelle développeur web, nous avons réalisé une application web permettant de créer et de visualiser sur une map le parcours d’un barathon (parcours de différents bars) avec ReactJS. Ce TP guidé m’a permis de découvrir ReactJS, TypeScript ainsi que NodeJS.

Description du projet

Voici ce que j’ai réalisé durant ce TP :
  • développement d’un serveur API en NodeJS pour interroger notre base de données (mongoDB)
  • développement d’une application web avec ReactJS
  • affichage d’une map, des marker, et l’itinéraire entre deux barathons avec leafletJS
Développement d’une petite API en NodeJS :
  • création des schémas de la base de donnée, connexion à la base
  • mise en place des différentes routes qui retournent les données en JSON
  • création d’un middleware “loggers” afin d’avoir les logs
Développement d’une application web avec ReactJS :
  • mise en place et configuration d’un projet ReactJS
  • création de divers components
  • utilisation de styled-components pour styliser les composants
Conclusion :

Ce projet m’a permis de découvrir rapidement le framework ReactJS, le langage de programmation TypeScript ainsi que NodeJS. Trois technologies que je n’avais jamais utilisées auparavant.

J’ai trouvé cette première expérience avec ReactJS très intéressante, mais peut-être un peu moins logique que VueJS. Au terme de ce TP, j’ai eu envie de découvrir un peu plus ReactJS. C’est donc ce que je compte faire en 2021.

En ce qui concerne typeScript, c’est un langage que j’ai trouvé assez barbare et cruel, mais j’ai compris son utilité dans des projets de grandes envergures. Avec un peu plus d'expérience, il est tout à fait possible qu’on devienne copains, pour l’instant, c’est pas trop ça.

Pour ce qui est de NodeJS, j’ai également beaucoup apprécié cette technologie.
