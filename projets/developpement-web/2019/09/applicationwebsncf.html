Un projet web

Conception d’une application web de gestion (réservation, stock) pour la SNCF Réseau avec Laravel

Dans le cadre de ma seconde année d’institut universitaire de technologie dans le domaine du multimédia et de l’internet, j’ai été en contrat apprentissage pour une durée d’un an à la SNCF, où j’ai réalisé, entre autres, une application web permettant de gérer la réservation d’engins.

Description du projet

Voici les principales tâches réalisées:
  • conception du design de l’application web en fonction de la charte graphique existante
  • réalisation des maquettes graphiques (zoning, wireframe puis prototype) avec adobe XD
  • conception de l’arborescence en fonction du besoin utilisateur
  • développement du projet en PHP avec le framework Laravel
  • retour d'expérience utilisateur
Conception du design de l’application web:

En fonction de la charte graphique de la SNCF, j’ai effectué certains choix graphiques afin d’avoir une interface agréable et moderne.

Intégration de l’application web:

Concernant l’intégration de ce design, j’ai décidé d'utiliser bootstrap et un template nommé “Gentelella” afin de gagner du temps. À cette période, je ne maîtrisais pas comme aujourd’hui l’intégration web, j’ai donc préféré concentrer le plus de temps possible sur le développement backend. Le fait d’utiliser Bootstrap m’a permis de gagner un temps considérable.

Développement du projet en PHP avec Laravel:

Je n’avais à ce jour jamais entendu parler de Laravel. Après m’être lancé dans le développement de cette application from-scratch avec quelques notions de PHP acquises à l’IUT et m’être ensuite confronté à divers problèmes de sécurité, d’optimisation, je me suis mis à chercher une alternative. Un collègue de travail m’a parlé de Laravel et m’a mis le pied à l’étrier.

Avec le soutien de collègues développeurs web, l’excellente documentation de Laravel ainsi que sa communauté très active, j’ai pu mener à terme le projet dans les temps.

Technologies et outils utilisés

Html5 Php Laravel
