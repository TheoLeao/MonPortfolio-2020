Un projet web

Interface de jeu de rôle en PHP from-scratch - authentification, création de parties

Dans le cadre de ma formation DUT - Métiers du multimédia et de l’internet, nous devions réaliser en groupe de 5 un jeu de rôle en PHP. Ce projet consistait à créer un jeu de rôle en PHP from-scratch avec quelques notions de programmation orienté objet vu en cours.

Description du projet

Ce que j’ai réalisé sur ce projet:
  • Structure html/css de base avec bootstrap pour se concentrer sur le backend
  • Modélisation UML de la base de données
  • Gestion de l’authentification (inscription, connexion, déconnexion)
  • Gestion de la partie (enregistrement en BDD de l’avancée de la partie: score, tour, joueurs)
  • Développement de la page contact en PHP

Technologies et outils utilisés

Html5 JS Php
